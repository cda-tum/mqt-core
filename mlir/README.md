# MQT MLIR - Interfacing the MQT with MLIR

This experimental library aims to create an interface between the [_Munich Quantum Toolkit (MQT)_](https://mqt.readthedocs.io), which is a collection of design automation tools and software for quantum computing developed by the [Chair for Design Automation](https://www.cda.cit.tum.de/) at the [Technical University of Munich](https://www.tum.de/), and the MLIR framework and its various Quantum Dialects.
